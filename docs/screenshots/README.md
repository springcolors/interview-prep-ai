**How to fix broken screenshots on GitHub**

1. Copy your 6 screenshots into this folder (`docs/screenshots/`).
2. Rename them to match the table above (e.g. `01-header.png`, `02-input-start.png`, …).
3. Commit and push:
   ```bash
   git add docs/screenshots/*.png
   git commit -m "Add README screenshots"
   git push origin main
   ```

Use PNG or JPG. Recommended width: 600–800px.
